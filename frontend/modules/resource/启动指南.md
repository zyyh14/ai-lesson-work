### 1. 前端已启动 ✅


前端运行在 `http://localhost:3000`

### 2. 启动后端服务 ⚠️

**重要**：前端需要后端 API 才能正常工作！

后端应该运行在 `http://localhost:8000`

请在后端项目目录执行：

```bash

python main.py

uvicorn main:app --reload --port 8000
```

### 3. 验证后端连接

打开浏览器访问：
- 后端健康检查: http://localhost:8000/health
- 后端 API 文档: http://localhost:8000/docs

应该看到：
```json
{"status": "healthy"}
```

## 🐛 常见问题

### 问题 1: 网络连接失败

**错误**: "网络连接失败，请检查后端服务是否启动"

**解决方案**:
1. 确认后端服务运行在 `http://localhost:8000`
2. 访问 http://localhost:8000/health 检查后端状态
3. 检查后端日志是否有错误

### 问题 2: 搜索资源超时

**原因**: 资源搜索需要 20-40 秒（搜索+爬取+AI整理）

**解决方案**:
- 耐心等待，查看加载提示
- 不要重复点击按钮
- 如果超过 60 秒无响应，检查后端日志

### 问题 3: 生成练习题失败

**可能原因**:
- 智谱 API 密钥无效
- API 额度不足
- 网络问题

**解决方案**:
1. 检查 `.env.local` 中的 `ZHIPU_API_KEY`
2. 登录智谱 AI 控制台检查额度
3. 查看后端日志获取详细错误信息

### 问题 4: 前端启动失败

**错误**: Node.js 版本不兼容

**解决方案**:
- 当前配置支持 Node.js 16+
- 如果仍有问题，升级到 Node.js 18+ LTS 版本

## 📊 API 接口说明

### 1. 搜索教学资源

```http
GET http://localhost:8000/api/v1/resources/search?query=静夜思&limit=5
```

**响应时间**: 20-40 秒

**返回数据**:
- 资源列表
- AI 整理的完整教学内容
- 参考资源链接

### 2. 生成练习题

```http
POST http://localhost:8000/api/v1/exercises/generate
Content-Type: application/json

{
  "knowledge_point": "初中水平 - 李白《静夜思》"
}
```

**响应时间**: 5-15 秒

**返回数据**:
- 选择题（带选项）
- 填空题
- 简答题
- 答案和解析
